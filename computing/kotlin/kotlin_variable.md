## TIPOS BÁSICOS

### BOOLEANOS

El tipo de dato `Boolean` es el encargado de almacenar un valor entre **verdadero** (`True`) y **falso** (`False`).

### NÚMEROS ENTEROS

El tipo de dato por defecto para los números enteros es el `Int`.
