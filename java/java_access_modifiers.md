# MODIFICADORES DE ACCESO

| MODIFICADOR   | CLASE | PAQUETE | SUBCLASES | MUNDO |
| ------------- | ----- | ------- | --------- | ----- |
| `public`      | YES   | YES     | YES       | YES   |
| `protected`   | YES   | YES     | YES       | NO    |
| *no modifier* | YES   | YES     | NO        | NO    |
| `private`     | YES   | NO      | NO        | NO    |
^access-modifiers-table
