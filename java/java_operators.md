# OPERADORES EN JAVA

## OPERADORES ARITMÉTICOS

| OPERADOR | NOMBRE |
|:--------:|:------ |
|    +     | x + y  |
|    -     | x - y  |
|    \*    | x \* y |
|    /     | x / y  |
|    %     | x % y  |
|    ++    | x++    |
|    --    | x--    |
^arithmetic-operators-table

## OPERADORES DE ASIGNACIÓN

| OPERADOR | EJEMPLO | IGUAL QUÉ  |
|:--------:|:------- |:---------- |
|    =     | x = 5   | x = 5      |
|    +=    | x += 5  | x = x + 5  |
|    -=    | x -= 5  | x = x - 5  |
|   \*=    | x \*= 5 | x = x \* 5 |
|    /=    | x /= 5  | x = x / 5  |
|    %=    | x %= 5  | x = x % 5  |
|    &=    | x &= 5  | x = x & 5  |
|   \|=    | x \|= 5 | x = x \| 5 |
|    ^=    | x ^= 5  | x = x ^ 5  |
|   >>=    | x >>= 6 | x = x >> 5 |
|   <<=    | x <<= 5 | x = x << 5 |
^assignment-operators-table

## OPERADORES DE COMPARACIÓN

| OPERADOR | NOMBRE            |
|:--------:|:----------------- |
|    ==    | Igual a           |
|    !=    | Diferente de      |
|    >     | Mayor qué         |
|    <     | Menor qué         |
|    >=    | Mayor o igual qué |
|    <=    | Menor o igual qué |
^comparison-operators-table
